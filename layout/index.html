India COVID-19 Data Archive

This website archives datewise case numbers for COVID-19 cases in India along with links to archived snapshots of official sources as supporting evidence.

This website can be also be reached using git.io/indiacovid19 (a short link).

JSON Data

Click on the following link to view/download the raw JSON data from which this page is generated:

indiacovid19.json

This JSON file contains a list of list entries. Each inner list entry is a septuple (7-tuple) consisting of 7 values. Here is an example entry: 

[ "2020-03-20", 196, 22, 4, 1, "2020-03-20 17:00",
  "https://web.archive.org/web/20200320173607/https://www.mohfw.gov.in/" ]

Here is a description of these 7 values in the order they appear in each septuple:

  1. Date (in "YYYY-MM-DD" format)
  2. Number of active cases
  3. Number of cured cases
  4. Number of death cases
  5. Number of migrated cases
  6. Last updated time as found in the source of the data (in "YYYY-MM-DD HH:MM" format)
  7. Reference archive link of the source of the data as supporting evidence

If we add the four numbers in an entry for a specific date, we get the total number of cases for that date.

In this page, the sum of the number of cured cases and that of the migrated cases is reported as the number of cured cases because the MoHFW website too clubs them together in the statewise status tables.

If you find discrepancies in the data, please create new issues to report them or send pull requests to the corresponding GitHub project.

Questions and Answers

  1. Why was this project created when other similar projects already exist?

    Different projects have different conventions for reporting the numbers. Some projects follow the latest updates appearing in news to estimate the case numbers in real time. This project does not attempt to provide case numbers in real time. Instead, this project is an attempt to document the case numbers from official sources only as much as reasonably possible. Further, for each entry of case numbers, a reference archive URL is provided that points to an archived snapshot of an official source like MoHFW or an unofficial source like WoM that supports the numbers used in this project.

    As a convention, this project attempts to archive the last updated case numbers found in an official source for each date.

  2. Are all case numbers used in this project from official sources only?

    Most numbers used in this project are from an official source like MoHFW. However, for some dates, no archived snapshot of the MoHFW website is available. For such dates, an unofficial source like WoM has been used.

  3. Where can I ask more questions or provide feedback about this project?

    To ask questions or provide feedback either post new issues to this project or send a tweet to the author.

  4. How is this website generated?

    This website is generated using a couple of Python scripts available at github.com/indiacovid19.

    The static site generator used for this project is a modified version of Sunaina Pai's makesite.py.
